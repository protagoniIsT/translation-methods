# Ваше сгенерированное приложение

В репозитории не должно быть исходников приложения. Поддерживаемые C++ расширения представлены в [`Makefile`](./Makefile).

## Сборка и запуск

Для сборки вам понадобится

* компилятор GNU GCC (версии 11 и выше) или LLVM Clang (версии 18 и выше);
* система сборки GNU make.

| Команда        | Комментарий                            |
|:---------------|:---------------------------------------|
| `make build`   | Сборка программы                       |
| `make clean`   | Очистка от артефактов сборки программы |
| `make rebuild` | См. `make clean` и `make build`        |
| `make run`     | Запуск сгенерированного приложения     |
