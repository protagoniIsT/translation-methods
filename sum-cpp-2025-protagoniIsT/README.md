# Тестовая лабораторная работа. Сумма чисел

Текущий статус тестирования GitHub Actions: [![CI/CD](../../actions/workflows/classroom.yml/badge.svg?branch=main&event=workflow_dispatch)](../../actions/workflows/classroom.yml).

## Задание

Необходимо реализовать программу, которая вычислит сумму двух чисел. Имя входного файла: **`input.txt`**. Имя выходного файла: **`output.txt`**. Гарантируется, что оба файла существуют.

**Формат входных данных**: в первой и единственной строке даны два числа, непревосходящих $10^{9}$.

**Формат выходных данных**: вывести сумму чисел.

## Сборка и запуск

Для сборки вам понадобится

* CMake (версии 3.11 и выше);
* компилятор GNU GCC (версии 11 и выше);
* система сборки GNU make.

| Команда        | Комментарий                              |
|:---------------|:-----------------------------------------|
| `make build`   | Сборка программы с заданными параметрами |
| `make clean`   | Очистка от артефактов сборки программы   |
| `make rebuild` | См. `make build` и `make clean`          |
| `make run`     | Запуск приложения                        |

> [!CAUTION]
> *Не меняйте* файл [`Makefile`](Makefile) и содержимое [`.github/`](.github/). Допускается ТОЛЬКО *добавление* разработанных исходных файлов в [`CMakeLists.txt`](CMakeLists.txt)
