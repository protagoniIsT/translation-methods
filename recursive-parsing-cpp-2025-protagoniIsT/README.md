## Сборка и запуск

Для сборки вам понадобится

* CMake (версии 3.11 и выше);
* компилятор GNU GCC (версии 11 и выше);
* система сборки GNU make;
* систему визуализации GraphViz (версии 2.42 и выше).

| Команда        | Комментарий                                                             |
|:---------------|:------------------------------------------------------------------------|
| `make build`   | Сборка программы с заданными параметрами                                |
| `make clean`   | Очистка от артефактов сборки программы, описаний GraphViz и изображений |
| `make rebuild` | См. `make build` и `make clean`                                         |
| `make run`     | Запуск приложения и генерация изображения по описанию `tree.dot`        |

> [!CAUTION]
> *Не меняйте* файл [`Makefile`](Makefile) и содержимое [`.github/`](.github/). Допускается ТОЛЬКО *добавление* разработанных исходных файлов в [`CMakeLists.txt`](CMakeLists.txt).
