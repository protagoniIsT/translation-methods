# Контекстно-свободная грамматика для описания заголовка функции Kotlin


## 1. Лексический анализатор
### Описание языка
Заголовок функции в Kotlin. Заголовок начинается ключевым словом “fun”, далее идёт имя функции, скобка, несколько описаний аргументов через запятую, затем может идти двоеточие и имя возвращаемого типа.

Пример: `fun printSum(a: Int, b: Int): Unit`

### Терминалы
* `fun`
* `name`
* `(`
* `)`
* `<`
* `>`
* `:`
* `,`

### Нетерминалы
* S — стартовый нетерминал (заголовок функции)
* P — параметр функции
* P', P'' — последовательность параметров функции
* T — тип параметра функции
* TA, TA' — аргументы generic-типа
* R — возвращаемый тип


### Исходная грамматика(без левой рекурсии, с правым ветвлением)
S ⟶ fun name(P') R      
P' ⟶ ε       
P' ⟶ P     
P' ⟶ P, P'    
P ⟶ name: T    
R ⟶ : T    
R ⟶ ε  
T ⟶ name TA  
TA ⟶ \<TA'>  
TA ⟶ ε  
TA' ⟶ T  
TA' ⟶ T, TA'  

### Грамматика после устранения правого ветвления (LL(1))

S ⟶ fun name(P') R    
P' ⟶ ε                
P' ⟶ PP''          
P'' ⟶ , PP''       
P'' ⟶  ε                          
P ⟶ name: T  
R ⟶ : T             
R ⟶ ε  
T ⟶ name TA  
TA ⟶ < T TA' >   
TA ⟶ ε  
TA' ⟶ , T TA'  
TA' ⟶ ε  

## 2. Синтаксический анализатор

| Нетерминал |    FIRST     |    FOLLOW        |
| ---------- | ------------ | ------------     |
| **S**      |   `fun`      |   `$`            |
| **P'**     |  `ε`, `name` |   `)`            |
| **P''**    |   `ε`, `,`   |   `)`            |
| **P**      |   `name`     |   `,`, `)`       |
| **R**      |   `ε`, `:`   |   `$`            |
| **T**      |  `name`      | `,`, `)`, `>`,`$`|
| **TA**     |  `<`, `ε`    | `,`, `)`, `>`,`$`|
| **TA'**    |  `,`, `ε`    |      `>`         |
